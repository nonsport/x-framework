​X-Framework — это высокотехнологичная модульная платформа для проведения разведки по открытым источникам (OSINT). Проект разработан sept1kie в тесном сотрудничестве с проектом Manifest.
​Инструмент сочетает в себе агрессивный "хакерский" дизайн, высокую скорость работы благодаря асинхронности и гибкость за счет системы внешних плагинов.
​🔥 Основные особенности
​🎨 Imba UI: Полностью обновленный интерфейс на базе библиотеки Rich с поддержкой таблиц, панелей и центрирования.
​🧩 Plugin System: Легко расширяемый функционал — просто закиньте .py скрипт в папку plugins, и он автоматически появится в меню.
​🕵️ Deep Dive Socials: Поиск по социальным сетям без использования платных API (используется продвинутый скрейпинг и подмена User-Agent).
​🚀 Auto-completion: Умная командная строка с автозаполнением и историей команд.
​🛡 Security: Ротация User-Agent для обхода блокировок и анонимизации запросов.
​🛠 Установка и запуск
​Для работы требуется Python 3.8+ и установленный Git.
Гайд:

​1. Клонирование репозитория
​Откройте терминал и выполните: 
git clone https://github.com/nonsport/X-Framework.git
cd X-Framework

2. Установка зависимостей 
​Установите все необходимые библиотеки одной командой:
pip install -r requirements.txt

3. Запуск фреймворка 
python main.py

 Структура проекта
​main.py — Ядро системы, анимация и интерактивное меню.
​apis.py — Модули поиска, логика скрейпинга и сетевые запросы.
​requirements.txt — Список зависимостей.
​🔐 Политика конфиденциальности и использования
​1. Сбор данных
​X-Framework не собирает, не хранит и не передает личные данные пользователей на сторонние сервера. Все результаты поиска (логи) сохраняются исключительно локально на вашем устройстве в файл x_results.json.
